GTC SLC (``esar_gtc``)
-------------------------
The `esar_gtc` function extracts Sxy/C2/T2 (for dual-pol), S2/C4/T4/C3/T3/C2/T2 (for full-pol) matrix elements from a ESAR GTC SLC data folder and saves them into a directory.
